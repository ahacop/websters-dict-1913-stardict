# websters-dict-1913-stardict

The [Webster's Revised Unabridged Dictionary](https://en.wikipedia.org/wiki/Webster%27s_Dictionary#1913_edition) (editor Noah Porter, Springfield, MA: C. & G. Merriam Co., 1913), packaged in [StarDict](https://en.wikipedia.org/wiki/StarDict) format. The work is in the public domain.

## Contents

`stardict-dictd-web1913-2.4.2.tgz` contains:

- `dictd_www.dict.org_web1913.ifo` — StarDict info file
- `dictd_www.dict.org_web1913.idx` — word index
- `dictd_www.dict.org_web1913.dict.dz` — dzip-compressed definitions

## Usage

Extract the archive into the dictionary directory of any StarDict-compatible reader:

```sh
tar -xzf stardict-dictd-web1913-2.4.2.tgz
```

Common destinations:

- GoldenDict: `~/.goldendict/dic/` (or add the directory under Edit → Dictionaries → Sources)
- KOReader: `koreader/data/dict/`
- StarDict: `~/.stardict/dic/` or `/usr/share/stardict/dic/`

## Source

Converted from the [dict.org web1913](https://www.dict.org/) database. Original text from Webster's Revised Unabridged Dictionary, 1913 edition.
